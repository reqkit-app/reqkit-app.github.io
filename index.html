ReqKit

A native HTTP client for Windows developers

Test and debug APIs without the overhead. ReqKit is built directly with Win32 APIs — no Electron, no runtime, no bloat. Instant startup, low memory, runs on any Windows 10 or 11 machine.

Download on Microsoft Store

Features

  • Send GET, POST, PUT, PATCH, DELETE and HEAD requests
  • Organize requests into collections, browse full request history
  • Custom headers, query parameters, and request body (JSON, XML, form)
  • Basic and Bearer authentication support
  • Environments with {{variable}} substitution in URLs, headers and body
  • Response body download to file
  • Dark, Light, and Auto theme — follows Windows system setting

Requirements

  • Windows 10 or Windows 11 (64-bit)
  • No additional runtime or framework required
